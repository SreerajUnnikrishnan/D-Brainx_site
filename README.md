🧠 BrainX – Cybersecurity Platform

BrainX is a cybersecurity-focused web application designed to help users report cyberattacks, connect with ethical hackers, and improve digital security awareness.

---

🚀 Features

- 🔐 Secure Login & Authentication
- 🧑‍💻 User Dashboard
- 💬 Chat System with Ethical Hackers
- 📊 Cyberattack Reporting System
- 🛡️ Hacker Portal for Security Experts
- 📈 Pricing & Subscription System

---

🛠️ Tech Stack

- HTML, CSS, JavaScript
- Firebase (Authentication & Database)
- GitHub Pages (Deployment)

---

📂 Project Structure

- "index.html" → Landing Page
- "auth.html" → Login / Signup
- "dashboard.html" → User Dashboard
- "chat.html" → Chat System
- "admin.html" → Admin Panel
- "hacker-portal.html" → Hacker Interface
- "style.css" → Styling
- "script.js" → Functionality

---

🎯 Purpose

The goal of BrainX is to bridge the gap between users and cybersecurity professionals by providing a platform where users can get help after cyber incidents and learn how to stay secure online.
